- ✅ UI حديث وجميل بـ gradients وanimations

## 🚀 كيف تبدأ؟

### خطوة واحدة فقط:

```bash
# 1. ضع DeepSeek API Key في .env
nano .env

# 2. شغّل الموقع
npm start

# 3. افتح المتصفح
http://localhost:3000
```

## 🌍 مشاركة عالمية

```bash
# ثبت localtunnel
npm install -g localtunnel

# شغل السيرفر
npm start &

# شغل localtunnel
lt --port 3000

# شارك الرابط مع أي حد!
```

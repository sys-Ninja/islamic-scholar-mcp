🚀 كيف تبدأ؟

1. احصل على DeepSeek API Key

سجل على platform.deepseek.com واحصل على API Key مجاناً

2. ضع الـ Key في ملف .env

nano .env # غيّر: AI_API_KEY=your_deepseek_api_key_here # إلى: AI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

3. شغّل الموقع

npm start # أو ./START.sh

4. افتح المتصفح

http://localhost:3000

🌍 مشاركة الموقع عالمياً

استخدم localtunnel (الأسهل)

# ثبت localtunnel npm install -g localtunnel # شغل السيرفر npm start & # شغل localtunnel lt --port 3000 # سيعطيك رابط مثل: # https://funny-cat-123.loca.lt

شارك الرابط مع أي حد في العالم! 🌍

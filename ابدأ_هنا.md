# 🕌 الشيخ الرقمي - ابدأ من هنا!

## 🎯 خطوات التشغيل (بسيطة جداً!)

### 1️⃣ احصل على DeepSeek API Key (مجاني!)

1. افتح المتصفح على: **https://platform.deepseek.com/**
2. سجل حساب جديد (مجاني)
3. اذهب إلى **API Keys**
4. اضغط **Create API Key**
5. انسخ الـ Key (يبدأ بـ `sk-`)

---

### 2️⃣ ضع الـ Key في ملف .env

افتح Terminal واكتب:

```bash
nano .env
```

ستجد السطر:
```
AI_API_KEY=your_deepseek_api_key_here
```

غيّره إلى:
```
AI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
```

(ضع الـ key اللي نسخته)

احفظ الملف:
- اضغط `Ctrl+O`
- اضغط `Enter`
- اضغط `Ctrl+X`

---

### 3️⃣ شغّل الموقع!

```bash
npm start
```

سيظهر:
```
╔══════════════════════════════════════════╗
║  Islamic Scholar MCP Server              ║
║  Running on: http://localhost:3000       ║
║  AI Provider: deepseek                   ║
║  Model: deepseek-chat                    ║

✅ MCP Client connected with 11 tools
```

---

### 4️⃣ افتح المتصفح

افتح المتصفح على:
```
http://localhost:3000
```

**تمام! الموقع شغال! 🎉**

## 🌍 عايز تشارك الموقع مع أصحابك؟

### الطريقة الأسهل: localtunnel

في Terminal جديد، اكتب:

```bash
# ثبت localtunnel (مرة واحدة فقط)
npm install -g localtunnel

# شغل localtunnel
lt --port 3000
```

سيعطيك رابط مثل:
```
your url is: https://funny-cat-123.loca.lt
```

**شارك هذا الرابط مع أي حد!** 🌍

## ❓ مشاكل؟

### "AI_API_KEY not configured"
✅ تأكد من ملف .env وأن الـ key صحيح

### "MCP Client غير متصل"
✅ أعد تشغيل السيرفر: `npm start`

### "Port 3000 already in use"
✅ غيّر البورت في .env:
```bash
echo "PORT=3001" >> .env
```
